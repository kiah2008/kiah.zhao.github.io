# 绑定域名

## Github repo配置

在对应repo的Settings的Pages,选择Custom domain, 并填写具体的域名地址(name.github.io).
如需要添加顶级域名,只需要添加WWW和@的CNAME.

- A 域名关联到一个IPv4地址
- CNAME 域名关联到另外一个域名

# 网站加速
